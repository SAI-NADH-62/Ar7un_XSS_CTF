Stored XSS

Definition: Stored XSS occurs when input is persisted (database, storage, localStorage) and later rendered unsafely to other users or sessions.

Impact: persisted payloads run whenever a user views the content - enabling mass session theft, persistent phishing, unauthorized actions (account takeover, admin backdoors), and long‑term data exfiltration.

Tip: Trigger an alert containing the text "STORED" to reveal the flag.
